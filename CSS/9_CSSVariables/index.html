CSS Variables Theme Switcher

About CSS Variables

CSS Variables (officially called Custom Properties) allow you to define reusable values that can be referenced throughout your stylesheet. They're especially useful for theming and maintaining consistency across your design.

Variables are defined using the --variable-name syntax and accessed using the var(--variable-name) function.

Benefits of Theme Switching

Implementing theme switching with CSS Variables offers several advantages:

  • Improved accessibility for users with different visual preferences
  • Reduced eye strain in different lighting conditions
  • Potential energy savings on devices with OLED screens
  • Enhanced user experience through personalization

How It Works

This example defines a set of variables in the :root selector for the light theme, then overrides those variables in the .dark-theme class. The theme can be toggled by adding or removing the class from the body element.

Additionally, a .system-theme class uses the prefers-color-scheme media query to automatically match the user's system preference.
